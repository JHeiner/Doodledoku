Sorry about this, but Chrome prevents doodling on certain pages. First, the URL scheme must be one of these:

http: https: file:

If it is anything else doodling is not allowed. Second, the Chrome Web Store prevents doodling on any of its pages. So even if the scheme is OK, the host can not be:

chrome.google.com

That means you can not doodle on the extension's install page. Nor on any of these pages:

New Tab Bookmark Manager History Extensions Settings Downloads etc.

You can, however, doodle on this page, so give it a try. If you want to learn more, right-click the pencil icon and choose Options.

I imagine a lot of people click on the pencil icon immediately after installing, wanting to try it out. And I'm sure it is a bit unsettling to be redirected to this page. I really wish there were some way to bend the rules so that Doodledoku could run on its own Web Store pages. But until the rules change all I can do is apologize and explain them to you.
